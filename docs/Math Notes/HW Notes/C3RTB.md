## Vocabulary
- #Relate-The-Bases use the base number $$b$$ to refactor exponential equations, eg.
  - $$b^{n}b^{m}=b^{n+m}$$
  - $$\frac{b^{n}}{b^{m}}=b^{n-m}=\frac{1}{b^{n-m}}$$
  - $$(b^n)^m=b^{nm}$$
  - $$b^{-n}=\frac{1}{b^n}\;\;\&\;\;\frac{1}{b^{-n}}=b^n$$
  - $$(ab)^n=a^{n}b^{n}\;\;\&\;\;\bigg(\frac{a}{b}\bigg)^{n}=\frac{a^n}{b^n}$$
  - $$\sqrt[n]{b}=b^{1/n}$$
