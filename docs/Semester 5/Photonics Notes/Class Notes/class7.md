# Metrology Lesson 7
- holography is making an interference pattern like a mach-zender interferometer but collecting the recombination on a photosensitive material to produce an image
- can create the hologram using computer 3d modeling
- computer generated hologram can be slowly changed to find the shape of an aspherical surface (acting like a lens from spherical->aspherical->spherical)
- overlapping two or more holograms can produce moire fringes (looks like interference)
- useful for vibrometry modes of complex structures or vibrations
- Fringe contrast is the relative intensity of a reference and test arm in an interferometer, 1 being the most clear and 0 being indistinguishable.
  - $$\mathcal{V}=\frac{2\sqrt{I_{ref}I_{test}}}{I_{ref}+I_{test}}$$
